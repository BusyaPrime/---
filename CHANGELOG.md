## [1.0.0] - 2026-02-26
### Завезли (Added)
- Ядро 2D Геометрии и хардкорный движок стэнсилов на примитивных массивах (zero-allocations, память не течет).
- Свой кастомный `ParallelExecutor`, чтобы обходить false-sharing ForkJoinPool-а и гарантировать детерминизм при сборке флоатов (никакой плавающей магии).
- Неявный Солвер (Implicit Solver) на базе PCG с прекондеем Якоби.
- Настраиваемый шаг по времени (завезли Crank-Nicolson и Backward Euler).
- Нормальное структурированное логирование через SLF4J (чтоб в консоли не было мусора).
- Реестр артефактов для 100% воспроизводимости (плюем JSON конфиги и трекаем метрики по феншую).
- Накатили бенчи JMH для замера пропускной способности памяти на Hot-циклах (выжимаем все из железа).
- Регрессионные и валидационные тесты MMS (Method of Manufactured Solutions) на базе JUnit 5.
- Блоки проверки CI/CD через Jacoco (coverage).
- Подцепили SLURM скрипты для масштабирования расчетов на кластерах HPC.
